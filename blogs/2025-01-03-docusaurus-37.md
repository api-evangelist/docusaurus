<p>We are happy to announce <strong>Docusaurus 3.7</strong>.</p>
<p>Docusaurus is now compatible with <a class="" href="https://react.dev/blog/2024/12/05/react-19" rel="noopener noreferrer" target="_blank">React 19</a>.</p>
<p>Upgrading should be easy. Our <a class="" href="https://docusaurus.io/community/release-process">release process</a> respects <a class="" href="https://semver.org/" rel="noopener noreferrer" target="_blank">Semantic Versioning</a>. Minor versions do not include any breaking changes.</p>

<h3 class="anchor anchorTargetHideOnScrollNavbar_qJOx" id="react-19">React 19<a class="hash-link" href="https://docusaurus.io/blog/releases/3.7#react-19" title="Direct link to React 19">​</a></h3>
<p>In <a class="" href="https://github.com/facebook/docusaurus/pull/10763" rel="noopener noreferrer" target="_blank">#10763</a>, we added support for <a class="" href="https://react.dev/blog/2024/12/05/react-19" rel="noopener noreferrer" target="_blank">React 19</a>, and the Docusaurus website is running on React 19 already.</p>
<p>From now on, all newly initialized sites will run on React 19 by default, and React 19 will be the minimum required version Docusaurus v4.</p>
<p>However, React 18 remains supported, and existing Docusaurus sites can either choose to stay on React 18, or upgrade their dependencies to React 19:</p>

<li class="">You have custom React code and want to ensure it is ready for React19</li>

<p>Along the way, we <a class="" href="https://github.com/facebook/docusaurus/pull/10786" rel="noopener noreferrer" target="_blank">fixed</a> all the remaining hydration errors reported by React 19, some of them produced by our aggressive HTML minifier settings.</p>

<h2 class="anchor anchorTargetHideOnScrollNavbar_qJOx" id="other-changes">Other changes<a class="hash-link" href="https://docusaurus.io/blog/releases/3.7#other-changes" title="Direct link to Other changes">​</a></h2>

<ul>
<li class=""><a class="" href="https://github.com/facebook/docusaurus/pull/10768" rel="noopener noreferrer" target="_blank">#10768</a>: Blog authors have built-in icons for social platforms bluesky, mastodon, threads, twitch, youtube, instagram.</li>
<li class=""><a class="" href="https://github.com/facebook/docusaurus/pull/10729" rel="noopener noreferrer" target="_blank">#10729</a>: Blog now supports <code>frontMatter.sidebar_label</code></li>
<li class=""><a class="" href="https://github.com/facebook/docusaurus/pull/10803" rel="noopener noreferrer" target="_blank">#10803</a>: <code>@docusaurus/remark-plugin-npm2yarn</code> now supports Bun conversions.</li>
<li class=""><a class="" href="https://github.com/facebook/docusaurus/pull/10672" rel="noopener noreferrer" target="_blank">#10672</a>: Upgrade Algolia DocSearch to <code>algoliasearch</code> v5.</li>
<li class=""><a class="" href="https://github.com/facebook/docusaurus/pull/10800" rel="noopener noreferrer" target="_blank">#10800</a>: Docusaurus Faster turns Rspack incremental mode on by default.</li>
<li class=""><a class="" href="https://github.com/facebook/docusaurus/pull/10783" rel="noopener noreferrer" target="_blank">#10783</a>: Improve Dutch theme translations.</li>

</ul>
